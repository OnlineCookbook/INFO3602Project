This project is a web-based recipe platform developed for the INFO3602 Web Programming course. The purpose of the website is to allow users to easily browse, search, and share recipes in a simple and interactive way. The idea behind the project was to create something practical that people can actually use, while also applying the concepts we learned in class.

The website includes several main sections such as a home page, a recipe listing page, individual recipe pages and a submission form where users can add their own recipes. Recipes are organized into categories like breakfast, lunch, dinner, and desserts, making it easier for users to find what they are looking for. Each recipe contains ingredients, step-by-step instructions, and images to guide the user.

In terms of functionality, the project uses WordPress with custom post types to manage recipe content. A search feature was also implemented to allow users to quickly find recipes without having to browse through everything manually. The site focuses on simplicity, so navigation is straightforward and easy to understand.

Throughout the development process, we worked on structuring the site properly and making sure all features functioned as expected. One of the main challenges was integrating different parts of the site while keeping the design consistent. However, through testing and adjustments, we were able to create a working system.

Overall, this project helped us gain a better understanding of how WordPress can be used to build dynamic websites. It also showed the importance of planning, organization, and teamwork when developing a web-based system.
